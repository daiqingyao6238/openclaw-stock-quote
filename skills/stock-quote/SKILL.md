# 股票实时行情

使用 `scripts/stock_quote.py` 脚本获取免费实时股票数据。

## 数据源

- **来源**: 腾讯财经 (qt.gtimg.cn)
- **特点**: 免费、无需API密钥、实时推送
- **支持**: A股（上海/深圳）、港股

## 股票代码格式

| 市场 | 代码前缀 | 示例 |
|------|----------|------|
| 上海A股 | sh | sh600519 (茅台) |
| 深圳A股 | sz | sz000001 (平安银行) |
| 港股 | hk | hk00700 (腾讯控股) |

也可直接输入6位数字代码，脚本会自动判断市场。

## 使用方式

### 单只股票

```bash
python scripts/stock_quote.py sh600519
# 或
python scripts/stock_quote.py 600519
```

### 批量查询

```bash
python3 scripts/stock_quote.py sh600519 sz000001 hk00700
```

## 返回字段说明

| 字段 | 说明 |
|------|------|
| name | 股票名称 |
| code | 股票代码 |
| current_price | 当前价格 |
| yesterday_close | 昨收价 |
| open | 开盘价 |
| high | 最高价 |
| low | 最低价 |
| volume | 成交量(手) |
| change | 涨跌额 |
| pct_change | 涨跌幅(%) |

## 注意事项

1. 数据仅供本人参考，不构成投资建议
2. 港股需要使用 hk 前缀
3. A股交易时间为工作日 9:30-15:00（非交易时间数据可能为收盘价）
